== Description ==

Mit Hilfe dieses Plugins ist es möglich deutsche rechtssichere Texte (für Shops) zu aktualisieren. Die Texte werden von Protected
Shops (http://www.protectedshops.de) bezogen (Abo vorausgesetzt).

Das Plugin wurde für das Wordpress Plugin wpShopGermany (http://wpshopgermany.maennchen1.de) entwickelt. Es kann auch ohne das Shop
Plugin von maennchen1.de genutzt werden.

Um diesen Dienst nutzen zu können ist ein Account bei Protected Shops nötig. Bei Einsatz von wpShopGermany erhält man einen 5%
Gutschein auf das erste Protected Shop Jahresabo.

== Installation ==

Zur Installation beachten Sie bitte folgende Schritte:

1. wpshopgermany-protectedshops.php in den Ordner "wp-content/plugins" laden, oder über das Wordpress Backend: Plugins >
Installieren > Hochladen, die ZIP-Datei hochladen, oder direkt über die Suchfunktion des Backends
2. Das Plugin im Plugin Menü von Wordpress aktivieren
3. Seiten für die Texte anlegen (im Wordpress Menü "Seiten")
4. Die Protected Shops Accountdaten in den Einstellungen des Plugins hinterlegen und speichern
5. Die Seiten auswählen auf denen die entsprechenden Texte angezeigt werden sollen (anschließend speichern)
6. Die Seiten bei denen die Texte aktualisiert werden sollen auswählen und speichern

Details unter http://wpshopgermany.maennchen1.de/2010/06/20/protected-shops/
